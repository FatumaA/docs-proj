To get started with the Unsplash API, you will need to complete the following steps:

<Steps>

1. Register for a [developer account](https://unsplash.com/developers).

2. Create an application within the developer account.

3. Get your access key and secret key.

</Steps>


The guide includes information on endpoints related to the following:

- [Get all photos](getphotos)
- [Search photos](searchphotos)
